RyzenAdj (libryzenadj.dll)
==========================

This plugin bundles `bin/libryzenadj.dll`, built from RyzenAdj.

How it is used
--------------
`libryzenadj.dll` is a separate, dynamically loaded shared library. Quick
Settings loads it at runtime via ctypes (see main.py, _load_ryzenadj). Quick
Settings' own code is licensed under the MIT License and is NOT a derivative
work of RyzenAdj; it merely calls the library's public C API. Under the LGPL,
this dynamic-linking arrangement is permitted, and the user is free to replace
the bundled DLL with their own build of the same library.

Written offer of corresponding source (LGPL-3.0 / GPL-3.0 sec. 6)
----------------------------------------------------------------
The complete corresponding source for the bundled `libryzenadj.dll`, including
the build scripts used to produce it, is publicly available at:

    https://github.com/FlyGoat/RyzenAdj

For a specific build, see the matching tag/commit in that repository.
